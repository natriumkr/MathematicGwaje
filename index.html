복소수란?

복소수는 실수와 허수로 이루어진 수입니다.

복소수의 형태: z = a + bi

· a는 실수부
· b는 허수부 (i² = -1)
· 예: 3 + 2i, -1 + 4i

  • 복소수는 2가지 정보를 담아요:
  • ① 크기: √(a² + b²)
  • ② 방향: 위상(각도)
  • 왜 필요한가? 양자 상태를 나타낼 때 확률의 크기와 위상 둘 다 필요하니까요!

🔗 중첩 상태란?

🎲 쉬운 비유로 이해하기

일반 동전: 떨어질 때 이미 결정됨 (앞면 또는 뒷면)

양자 동전: 떨어지는 도중에는 앞면과 뒷면이 「동시에」 존재
손으로 잡는 순간(= 측정) 앞면 또는 뒷면으로 결정됨

→ 측정 전: 0과 1이 함께 존재 (중첩)

→ 측정 후: 0 또는 1로 확정

→ 각 확률: 복소수로 표현

🌊 간섭 현상

물결의 간섭처럼...

연못에 돌을 던지면: 파동이 퍼집니다.

• 두 파동이 같은 방향 → 더 커짐 (건설적 간섭)
• 두 파동이 반대 방향 → 서로 상쇄 (상쇄적 간섭)

양자컴퓨터는 이 원리를 이용합니다:

✓ 정답의 확률은 더 크게 (건설적 간섭)
✓ 오답의 확률은 작게 (상쇄적 간섭)
✓ 복소수의 위상(방향)이 간섭을 조절합니다!

양자컴퓨터의 강점

왜 양자컴퓨터가 빠를까?

2큐비트: 동시에 4가지 상태
(2 × 2 = 4)

3큐비트: 동시에 8가지 상태
(2 × 2 × 2 = 8)

300큐비트: 동시에 2³⁰⁰가지!
(일반컴퓨터로는 불가능)
